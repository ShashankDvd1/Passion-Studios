# Approval Levels
## 1. Vision
Approval levels scale with an agent's Autonomy Level and the task's Priority.

## 6. Requirements
### 6.1 Autonomy-Based Approvals
| Agent Autonomy | Required Approval for P2/P3 Tasks |
|----------------|-----------------------------------|
| **L1 (Supervised)** | Full Quality Pipeline (Self → Technical → Consistency) + Founder final check initially. |
| **L2 (Guided)** | Self-Review + Technical Review. |
| **L3 (Independent)** | Self-Review only (random audits applied). |
| **L4 (Autonomous)** | None. Direct execution. |

### 6.2 Priority-Based Approvals
Regardless of Autonomy Level, **P0 and P1 decisions always require Founder approval** (or CEO Agent during unavailability).
