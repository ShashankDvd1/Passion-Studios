# Responsibility Matrix
## 1. Vision
Standard RACI (Responsible, Accountable, Consulted, Informed) modified for AI agents.

## 6. Requirements
### 6.1 The RASI Framework
- **Responsible:** The agent doing the actual work (e.g., Gameplay Programmer Agent).
- **Accountable:** The agent (or Founder) who owns the quality of the outcome (e.g., Unity Architect Agent).
- **Supporting:** Agents providing necessary inputs or validations (e.g., QA Agent).
- **Informed:** Agents who need to know the outcome to do future work (e.g., Analytics Agent).

### 6.2 Implementation in Workflows
Every step in a Workflow (`10-NNN`) implicitly follows this structure based on its `Inputs`, `Outputs`, and `Quality Gates`.
