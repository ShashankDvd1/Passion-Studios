# Persistent Company Memory
## Mission
Store company-level knowledge that applies to all games, all agents, all time. This is the foundation context injected into every agent interaction.
## Content Categories
| Category | What's Stored | Expiry | Examples |
|----------|-------------|--------|---------|
| **Values & Principles** | Core values, design principles, operating principles | Permanent | "Worlds Over Games," "Automate First" |
| **Strategic Direction** | Current company vision, priorities, North Star | Permanent (updated quarterly) | "Focus on cozy simulation," "HRP is North Star" |
| **Brand Identity** | Studio personality, tone of voice, visual identity | Permanent | "Whimsical, Caring, Crafted" personality traits |
| **Ethical Boundaries** | Prohibited/permitted practices | Permanent | "No loot boxes, no pay-to-win" |
| **Company Milestones** | Key achievements and dates | Permanent | "Game 1 soft launch: Q4 2026" |
| **Lessons Learned** | Company-level insights from post-mortems | Permanent | "Variable reward > fixed reward for D7 retention" |
## Access: All agents have read access. Documentation Agent and Founder have write access.
## Context Injection: First 500 words of every agent context injection comes from Company Memory.
