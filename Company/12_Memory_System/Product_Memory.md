# Product Memory
## Mission
Store product-level context: backlog state, roadmap evolution, prioritization history, and feature pipeline. Provides the "what are we building and why" context.
## Content Categories
| Category | What's Stored | Expiry |
|----------|-------------|--------|
| **Backlog State** | Current prioritized backlog with RICE scores | Continuously updated |
| **Roadmap** | Current quarterly and annual roadmap | Updated quarterly |
| **Prioritization History** | Past prioritization decisions and outcomes | Project lifetime |
| **Competitor Responses** | Features built in response to competitive moves | Project lifetime |
| **User Requests** | Aggregated player feedback themes | 12 months rolling |
| **Feature Pipeline** | Ideas in various stages of evaluation | Until shipped or killed |
## Injected when: Product agents prioritize, plan, or evaluate features.
