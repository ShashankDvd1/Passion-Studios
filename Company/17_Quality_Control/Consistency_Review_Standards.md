# Consistency Review Standards
## 1. Vision
A feature might be technically perfect but feel completely wrong for the game. Consistency reviews ensure alignment with the world, the brand, and the economy.

## 6. Requirements
### 6.1 Review Dimensions & Reviewers
- **Ethical/Behavioral:** Behavioral Psychology Agent checks for dark patterns and anti-emotions.
- **Economic:** Economy Agent checks for sink/faucet balance and inflation risks.
- **Visual/Tone:** Art Director / Content Creator checks alignment with Studio Identity.

### 6.2 The Consistency Test
The core question of this review stage: "Does this feel like it belongs in Wonder Forge Studios' games, or does it feel like a generic mobile game feature?"

### 6.3 Handling Rejections
Consistency rejections are often subjective. The reviewer must provide specific examples of *why* it feels misaligned and reference a specific core value or design principle.
