# Game Mechanic Workflow
## Trigger
Manual — Game Designer or Product Manager identifies a new mechanic need.
## Steps
1. **Mechanic Concept** (Game Designer) → One-page concept with target emotion
2. **Behavioral Validation** (Behavioral Psychology Agent) → Ethical assessment, engagement model
3. **Economy Impact** (Economy Agent) → Resource flow analysis
4. **Feasibility Review** (Unity Architect Agent) → Technical feasibility and reusability assessment
5. **Design Spec** (Game Designer) → Full mechanic specification
6. **Founder Approval** (Founder) → P1 governance gate
## Estimated Duration: 3-5 days
