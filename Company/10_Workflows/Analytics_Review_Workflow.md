# Analytics Review Workflow
## Trigger
Scheduled — Weekly (every Monday).
## Steps
1. **Data Pull** (Analytics Agent) → Aggregate weekly KPIs from all data sources
2. **Trend Analysis** (Analytics Agent) → Week-over-week trends, anomalies, cohort analysis
3. **Insight Generation** (Analytics Agent) → Actionable insights with recommended actions
4. **Product Impact Assessment** (Product Manager Agent) → Map insights to feature backlog
5. **Strategic Review** (CEO Agent) → Company health scorecard update
6. **Founder Brief** (Founder Agent) → Include key insights in next morning brief
## Governance: P2 for standard reports. P1 if KPIs enter Red zone.
## Estimated Duration: 4-6 hours
